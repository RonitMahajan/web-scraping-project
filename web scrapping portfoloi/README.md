# Web Scraping & Price Analysis Project

## Overview
This project demonstrates end-to-end web scraping and data analysis using Python.

## Tools Used
- Python
- BeautifulSoup
- Selenium
- Pandas
- Matplotlib

## Features
- Static website scraping
- Dynamic content scraping
- Pagination handling
- Price tracking system
- Data visualization

## Projects
1. Book Store Scraper
2. Selenium Product Scraper
3. Amazon-style Dynamic Scraper
4. Automated Price Tracker

## Outcome
Collected and analyzed 1000+ product records to identify pricing and rating trends.
